提示语#

最后，我们给出一些提示，可以帮助你选择编译器和编辑器或 IDE。

  • 查看 此站点 以全面比较 Fortran 编译器。

  • 自 NVFortran 版本 17.1 起，已删除对计算能力 2.0 (Fermi) 及更低版本的支持。这很重要，因为在任何时候，只有最新的编译器版本可以免费下载。使用较旧 GPU 的用户将无法使用当前 (20.7) 或未来版本运行 CUDA Fortran。阅读 wiki 以了解你的 GPU 的计算能力，并与支持它的最新 CUDA SDK 进行交叉检查。

  • 有许多在线 Fortran 编译器。这些网站提供基本的文本编辑器，允许你在远程服务器上编译和运行代码。它们对于快速原型设计和测试特别有用。尽管我们承诺中立，但我们建议你查看开源和免费的 Compiler Explorer，这是一个具有独特功能的惊人工具。

  • 根据最近的在线调查，可以说最受欢迎的文本编辑器是 Microsoft 开发的 Visual Studio Code。

  • 最受欢迎的免费 IDE 是 Code::Blocks 和 Geany。许多商业 IDE 放弃了长达 30 天的免费试用期。请记住，商业 IDE 的价格可能会有所不同，有些可能相当实惠。最后，如果你是学生、开源开发人员或……爱好者 Fortraner，请不要犹豫与这些公司联系并申请折扣。在某些情况下，例如，在 R/Fortran 中，来自至少一家公司的代表向寻求负担得起的完整解决方案（IDE + 编译器）的个人提供折扣代码。

  • 最后，请加入我们的 Fortran Discourse 并毫不犹豫地发布你的问题并索取更多信息。
